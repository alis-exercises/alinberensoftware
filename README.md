# ALINBEREN SOFTWARE SOLUTIONS AND IT CONSULTANCY LTD

Premium SaaS-style consultancy website built with Next.js, Tailwind CSS, and Framer Motion.

## Tech

- Next.js (App Router, TypeScript)
- Tailwind CSS
- Framer Motion
- Space Grotesk + Inter fonts

## Run locally

```bash
npm install
npm run dev
```

## Static build

This project is configured for static export:

```bash
npm run build
```

The static site is generated in `out/`.

## GitHub Pages

This repository includes `.github/workflows/deploy-pages.yml` to build and deploy the exported site on every push to `main`.
